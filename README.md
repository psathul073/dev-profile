# Developer Profile

A simple and elegant developer profile page, similar to Linktree, that showcases your projects, source code demos, and social media links. This static page is fully responsive and visually appealing.

## Features
- Display personal projects with source code and live demos
- Social media buttons (Instagram, GitHub, Email, etc.)
- Clean and responsive design for all devices
- Built using React, Vite, and Tailwind CSS for fast performance and easy customization

## Technologies Used
- React
- Vite
- Tailwind CSS

## How to Use
1. Clone this repository:
   ```bash
   git clone git@github.com:psathul073/dev-profile.git
   ```
2. Navigate to the project directory:
   ```bash
   cd dev-profile
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
5. Customize the profile information, project links, and social media buttons in the `src` directory.
6. Can update project details.
